# usage of vi/vim under windows:

## move line

* up --> k

* down -->j

* left -->h

* right -->l

* linehead-->0

* noneBlankLineHead--> shift+^

* linetail-->shift+$

* up one line --> ctrl +e

* down one line --> ctrl +y

* to N line ---> :n    or-->nG

----

## move word

* word before  start--->w

* 				end --->e

* word after after -->b

* 				end --> ge

* to somechar  left-->Tx

* 				right--->fx

---

## move sentence 

* to start --> (

* to end --> )

---
## move paragraph

* to start {

* to end }

---
## move doc

*head --> gg
*end ---> G
*parecent -->50%

----

## screen move

* up a screen --> ctrl +b

* up half a acreen ---> ctrl + d

* down a screen ---> ctrl+f

* down half a screen --> ctrl +u

* to screen top -->H    3H

* to sreen middle --> M

* to screen button --> L 5L

* zt  --> cursor to screen top 

* zz --> cursor to screen middle

* zb --> cursor to screen buttom
